Electronic Components

I am using this project to create my first static webpage using HTML and CSS. Electronic Components Description helps you understand the core building blocks of modern electronics by providing clear and simple explanations of components such as resistors, capacitors, and more. To make learning easier and more engaging, this website also includes Tamil explanation videos and images, aiming to help students, and professionals learn electronics in an easy way.

AgriDirect

I am currently making a full-stack project, which is my second project, designed to support farmers. This platform helps eliminate middlemen by enabling farmers to sell their vegetables and other produce directly to buyers. The frontend is built using HTML and CSS, while FastAPI is used for backend development and database connectivity. By removing intermediaries, this project aims to help farmers earn higher profits and improve their livelihoods.
